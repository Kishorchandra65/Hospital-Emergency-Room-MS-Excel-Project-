📌 Project Overview

This project is an interactive Excel dashboard created to analyze hospital patient data and generate insights related to patient visits, waiting time, satisfaction scores, and admission status.

The dashboard helps visualize important metrics using Pivot Tables, KPI cards, and charts to support better decision-making in healthcare management.

🛠 Tools & Skills Used

Microsoft Excel

Pivot Tables

Pivot Charts

KPI Cards

Data Aggregation

Conditional Formatting

Data Visualization

📈 Key KPIs in the Dashboard

The dashboard tracks important healthcare metrics such as:

Total Number of Patients

Average Patient Wait Time

Average Patient Satisfaction Score

Admitted vs Not Admitted Patients

Overall Patient Experience Metrics

These KPIs help understand hospital performance and patient service quality.

📊 Dashboard Features

The Excel dashboard includes:

1️⃣ Patient Count Analysis

Shows the total number of patients visiting the hospital within the dataset.

2️⃣ Patient Wait Time Analysis

Calculates the average waiting time patients experience before treatment.

3️⃣ Patient Satisfaction Score

Displays the average satisfaction rating given by patients.

4️⃣ Admission Status Analysis

Breaks down patients into:

Admitted

Not Admitted

This helps analyze hospital admission trends.

5️⃣ Pivot Table Analysis

Pivot tables are used to summarize:

Patient count

Satisfaction scores

Wait time

Admission status
